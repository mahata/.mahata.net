What is Vagrant?

• (sort of...) VirtualBox F/E

Vagrant is F/E for various environments

• VirtualBox

• VMWare

• AWS

• DigitalOcean

• ...and more!

Vagrant manages VMs with Vagrantfile

• Minimal Example

Vagrant.configure("2") do |config|
  config.vm.box = "ubuntu-trusty"
  config.vm.box_url = "https://cloud-images.ubuntu.com/vagrant/trusty/current/trusty-server-cloudimg-amd64-vagrant-disk1.box"
end

Vagrant basic commands

$ vagrant up
$ vagrant halt
$ vagrant destroy
$ vagrant status
$ vagrant provision
$ vagrant reload

What is Ansible?

• A provisioner for machines

• Competitors: Chef, Puppet, SaltStack

Why provisioners?

• Write configurations as code

• Configurations with VCS (such as Git)

Why Ansible?

• No DSL is needed

• No Agent process is needed

• Configuration as YAML

• Support from RedHat
